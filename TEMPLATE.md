# Template: Landing Page

Marketing page with hero, features, pricing, and CTA

## Style and feature targets

Create a high-converting landing page. Requirements:
- Hero: bold headline (max 8 words), subheadline (1 sentence), primary CTA button with contrasting color, subtle background gradient or pattern
- Social proof bar: "Trusted by X+ users" or logos strip (use placeholder logos)
- Features: 3-4 cards in a grid, each with an SVG icon, short title, and 1-line description. Use CSS hover effects (scale + shadow transition)
- How It Works: 3-step horizontal flow with numbered circles and connecting lines
- Testimonials: 2-3 cards with quote, name, role, and avatar placeholder
- Pricing: 2-3 tier cards, highlight the recommended plan with a badge and border accent
- CTA banner: full-width section with headline and button before footer
- Footer: 3-column grid with links, social icons (SVG), and copyright
- Style: system font stack, 60fps scroll animations using IntersectionObserver (fade-up on scroll), mobile-first responsive, accessible contrast ratios, smooth anchor scrolling
- Dark mode support via prefers-color-scheme media query

## Suggested features

- Hero with CTA
- Social proof
- Features grid
- How it works
- Testimonials
- Pricing tiers
- Footer
